# @juscash/frontend

Frontend app for the technical case monorepo.

## Scripts

- `pnpm --filter @juscash/frontend dev`
- `pnpm --filter @juscash/frontend lint`
- `pnpm --filter @juscash/frontend test`
- `pnpm --filter @juscash/frontend typecheck`

## Notes

- App Router without locale in the URL.
- Default internal locale is `pt-BR`.
- Current auth screens are shell-only and do not perform real authentication yet.
